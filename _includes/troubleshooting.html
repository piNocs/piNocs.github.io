Troubleshooting

  • If you see a blue screen

    Unplug the Pi and plug it back in.

  • If you see a black screen

    Make sure the Pi is plugged in and a red light is lit on the bottom-right corner of the chip(ignore the green light). If the red light is on, flip the switch on top right side of the screen.

    If this doesnâ€™t work, make sure the 3 connections to the screen — 2 small wires and one bigger cable — are attached. If you followed all these steps and nothing has happened, look at the final troubleshooting issue.

  • The Pi is plugged in, but no red light is showing on the bottom-right corner of the Pi.

    Make sure there wasn't a short on that outlet. If the outlet works, then this means the Pi doesn't. The Pi will need to be replaced, but everthing else can be still be used.

  • The aforementioned red light is on, but the Pi still doesn't work

    You will need to remove the SD card and install my program onto it again. If you need a different video than the original, you will need to run through all the steps under the adding a new video section.
