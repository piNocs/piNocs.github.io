Developer Info

About the script

The script runs from /etc/rc.local. It consists of a few lines that run a loop on omxplayer.

The script shuts down the Pi at 5:25pm EST. The timezone must be set to Eastern Standard Time in raspi-config.

The video must be saved in /home/pi as video.mp4.

Accessing the Pi

The program runs off of rc.local for simplicity. To access the pi, you will need to connect via SSH. The Pi is already configured for this, just follow the directions in the link.

Once given access, enter sudo nano /etc/rc.local. Hash out the program I wrote and reboot. You can now play with the Pi!

Reinstalling the backup

If you need to reinstall the operating system and my program, get in touch with me. You will need to plug the SD card into your computer and we will troubleshoot from there.
