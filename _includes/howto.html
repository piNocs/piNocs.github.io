How to use the Pi-noculars

The Raspberry Pi is programmed to run a video on a loop as soon as it is plugged in. A after running for nine hours, the Pi shuts off on it's own.

That's it! Plug it in and it works. The Pi is plugged into a timer and should maintain itself. If people are seeing a blue screen, unplug it then plug it back in. If you ever find yourself handling the device, please be gentle.

Only unplug the device if you see a blue screen! By unplugging it unexpectedly, the SD card could become corrupt and the program will need to be reinstalled. If you find yourself needing to turn off the device, check out the shutdown instructions.
